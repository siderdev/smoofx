installation

Il y a deux moyens d'installer Smoofx : la version "prêt à consommer" et la version "à compiler". Seule la deuxième est configurable.
Smoofx nécessite jQuery v3.4.1 minimum.

version "prêt à consommer"

Cette version n'est pas configurable et nécessite d'être utiliser telle quel.

Smoofx n'est constitué que d'un seul fichier CSS et d'un seul fichier JS :

  • le CSS minifiée
  • et le JS minifiée

Pour information, Autoprefixer est activé sur cette version de KNACSS et la liste des navigateurs supportés (Browserslist) est 

[
      '> 1%',
      'last 2 versions',
      'IE >= 10', 'Edge >= 16',
      'Chrome >= 60',
      'Firefox >= 50', 'Firefox ESR',
      'Safari >= 10',
      'ios_saf >= 10',
      'Android >= 5'];

version "à compiler"

Le fichier source principal est src/assets/sass/styles.scss, c'est lui qui se charge de l'import des vendors, variables, mixins et tous les fichiers de styles du projet. C'est ce fichier qu'il faut compiler.

KNACSS est compiler avec Gulp (les fichiers gulpfile.js et package.json sont fournis).

Les fichiers de configuration _config/_variables.scss et _config/_mixins.scss devront être copiés dans votre propre dossier de travail (et modifiés avec vos propres valeurs) afin de ne pas être écrasés lors de chaque mise à jour de KNACSS via NPM.

KNACSS fourni un Pense-bête en PDF pour vous souvenir des classes utiles.
